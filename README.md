Bank Locker System:
A simple web app to help banks manage safety deposit lockers digitally.

What it does:
Find lockers - Search by number or customer name
Assign lockers - Register new customers to lockers
Track everything - See all locker status at a glance
No more paper registers - everything's in one place.

Built with:
Frontend: HTML, CSS, JavaScript

Backend: PHP, MySQL

Tools: XAMPP for local testing

Quick start:
Install XAMPP
Clone this repo to C:/xampp/htdocs/
Start Apache & MySQL from XAMPP
Visit http://localhost/Bank-Locker
